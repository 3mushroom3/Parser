📄 Пользовательское соглашение

1. Общие положения

Настоящее соглашение регулирует условия использования информационно-аналитического сервиса «Реестр производителей зерна» (далее — Сервис). Используя Сервис, вы принимаете условия настоящего соглашения в полном объёме.

2. Источник данных

Все сведения о декларациях соответствия, представленные в Сервисе, получены исключительно из открытого государственного реестра Федеральной службы по аккредитации (Росаккредитация) — официального портала pub.fsa.gov.ru, функционирующего в соответствии с Федеральным законом № 412-ФЗ «Об аккредитации в национальной системе аккредитации».

Указанные данные являются общедоступными и размещены государственным органом в открытом доступе для неограниченного круга лиц. Сервис не является официальным ресурсом Росаккредитации и не аффилирован с ним. Сервис выполняет функцию агрегатора и инструмента удобного поиска по открытым данным.

3. Актуальность данных

Данные периодически обновляются из источника pub.fsa.gov.ru. Сервис не несёт ответственности за расхождения между отображаемой информацией и актуальным состоянием реестра Росаккредитации. Для получения юридически значимых сведений следует обращаться непосредственно к официальному реестру.

4. Условия использования

Пользователь обязуется использовать Сервис исключительно в законных целях. Запрещается: автоматизированный сбор данных, попытки несанкционированного доступа, распространение данных в коммерческих целях без согласия правообладателя.

5. Подписка и оплата

Доступ к расширенному функционалу Сервиса предоставляется на платной основе. Подписка активируется после успешного прохождения оплаты. Возврат средств за неиспользованный период подписки не предусмотрен.

6. Ограничение ответственности

Сервис предоставляется «как есть». Администрация не несёт ответственности за достоверность сведений, полученных из открытого реестра pub.fsa.gov.ru, а также за возможные перебои в работе Сервиса, вызванные техническими причинами или изменениями в работе источника данных.

7. Персональные данные

При регистрации сохраняется только логин и хеш пароля. Сервис не собирает и не передаёт персональные данные пользователей третьим лицам. Обработка персональных данных осуществляется в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».
